# Worker Contract System

Projeto desenvolvido em Java para gerenciamento de trabalhadores, departamentos e contratos por hora, com cálculo de renda mensal.

---

## 📌 Objetivo

Simular um sistema simples de uma empresa que controla seus funcionários e contratos, calculando a renda mensal de cada trabalhador.

---

## ⚙️ Funcionalidades

- Cadastro de departamento
- Cadastro de trabalhador
- Associação de contratos por hora
- Cálculo de renda mensal baseado em:
  - Salário base
  - Contratos do mês selecionado

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Composição de classes
- Enumerações (Enum)
- Listas (ArrayList)
- Manipulação de datas (Calendar / Date)
- Encapsulamento

---
